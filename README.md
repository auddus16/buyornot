# 살까말까(buyornot)
2022 상명대학교 Elastic Training Program Team Project
### 중고거래 가격 비교 서비스
중고거래 플랫폼 "중고나라, 번개장터, 당근마켓"에서 Crawling하여 데이터 수집 후 Elasticsearch를 활용하여 판매자와 구매자에게 중고거래 가격 비교 서비스를 제공한다.
